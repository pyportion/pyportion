# Introduction

This guide will help you to get started in PyPortion, run your first commands and create your first project in few minutes.

## Installation

<div class="termynal" data-termynal>
    <span data-ty>pip install pyportion</span>
</div>

## Create your first project

At the beginning you need to download a template of your choice:

<div class="termynal" data-termynal>
    <span data-ty>portion template download flask</span>
</div>

Create a new project using `flask` template:

<div class="termynal" data-termynal>
    <span data-ty>portion new flask myapp</span>
</div>

This will create `myapp` folder by using `flask` template.

## Build a Portion

Using the built-in build command:

<div class="termynal" data-termynal>
    <span data-ty>portion build blueprint</span>
</div>

This will build `blueprint` portion based on the template instructions
